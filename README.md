# Watan Freelance System

Watan Freelance is a web-based freelance platform developed specifically for UKM students to support multimedia and technology-based freelance services. 
Demo Link: https://youtu.be/9voZ3CYhuCk?si=mZLgKKU3IWm-iqYm
## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Features
- User registration and login
- Freelancer profiles
- Service listings
- Task management system
- Communication and chat features
- Payment proof upload
- Feedback and review system

## Project Purpose
This project was developed to provide flexible freelance opportunities for UKM students while helping them gain real-world working experience and improve their professional skills. :contentReference[oaicite:2]{index=2}

## Development Methodology
- Agile Methodology
- Scrum Framework

## System Focus
The platform focuses on multimedia and technology-related freelance services such as:
- Graphic Design
- Video Editing
- Web Development
- Programming
